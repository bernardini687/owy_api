# OpenWeatherYelp API

### environment variables

refer to [.env.example](.env.example)

### setup

```sh
npm install
npm run start
```

### other scripts

```sh
npm run fix        # run `standard` to check for problems and fix them if found
npm run test
npm run test:watch # keep test running and auditing for file changes
```
